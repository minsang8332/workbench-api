# workbench-api
> 업무 향상을 위한 클라우드 API

## 목적
업무작업대 앱에서 파일로 보관중인 데이터를 클라우드로 보관하기 위함 <br>
그 외, PHP 블로그나 여타 다른 프로젝트를 Java/Spring 으로 이식하려 함.
